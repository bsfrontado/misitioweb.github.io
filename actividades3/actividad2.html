background-color - color de fondo

si no se especifica un color de fondo, el fondo es transparente.

css trata cada elemto html como un cuadro y la prioridad background-color establece el color del fondo para cuadro. se puede espicificar un color de fondo de las mismas tres formas en que puede especificar colores de primer plano: valores rgb, codigo hexadecimal y nombre de colores. tambien hemos utilizado la propiedad padding para separar el texto de los bordes de los cuadros esto hace que sea mas facil de leer
